Form Building Tutorial

Form Structure

The form is wrapped in a container with a header line:

<div class="form-header"></div>

This creates the simple 2px solid line at the top.

Stacked Name Fields

First and last name are in separate form groups:

<div class="form-group"> <label class="centered-label">First Name:</label> <input type="text"> </div> <div class="form-group"> <label class="centered-label">Last Name:</label> <input type="text"> </div>

Each has its own container for vertical stacking.

Status & Date Row

We use a flex container for side-by-side fields:

<div class="form-row"> <div class="form-group"> <label class="centered-label">Status:</label> <select>...</select> </div> <div class="form-group"> <label class="centered-label">Date:</label> <input type="date"> </div> </div>

The .form-row class creates the horizontal layout.

Left-Aligned Comments

For the comments section, we use a left-aligned label:

<div class="form-group"> <label class="left-label">Comments:</label> <textarea rows="4"></textarea> </div>

The .left-label class overrides the default centered alignment.
